# Чат-бот для Башкирэнерго

Этот проект представляет собой веб-приложение с чат-ботом поддержки для Башкирэнерго, разработанное на Vue.js. Приложение включает в себя систему аутентификации, чат-бота с интеграцией в существующий сайт и поддержку базы данных через Supabase.

## Функциональность

- Регистрация и авторизация пользователей
- Чат-бот поддержки с возможностью общения
- Интеграция с API Башкирэнерго
- Адаптивный дизайн в стиле сайта Башкирэнерго

## Технологии

- Vue 3 + Vite
- Vue Router 4
- Supabase (аутентификация и база данных)
- Axios (для API запросов)
- CSS (стилизация)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` в корне проекта и добавьте переменные окружения:
```env
VITE_SUPABASE_URL=ваш_supabase_url
VITE_SUPABASE_ANON_KEY=ваш_supabase_anon_key
VITE_BASHKIRENERGO_API_URL=ваш_api_url
```

3. Запустите приложение в режиме разработки:
```bash
npm run dev
```

## Интеграция с существующим сайтом

Для интеграции чат-бота в существующий сайт Башкирэнерго используйте компонент `ChatbotWidget.vue`, который можно встроить на любую страницу. Компонент представляет собой плавающую кнопку чат-бота, которая появляется в правом нижнем углу страницы.

## Структура проекта

- `src/App.vue` - основной компонент приложения
- `src/router/index.js` - маршруты приложения
- `src/views/` - страницы приложения
- `src/components/` - переиспользуемые компоненты
- `src/lib/supabaseClient.js` - клиент для работы с Supabase
- `src/style.css` - глобальные стили

## Настройка WebStorm

Для работы с проектом в WebStorm:

1. Откройте проект как Vue.js проект
2. Установите плагины Vue.js и Vite
3. Настройте поддержку ES6+ и SFC (Single File Components)
4. Для работы с dev-контейнером настройте Docker integration

## Supabase

Для работы с Supabase:

1. Зарегистрируйтесь на https://supabase.com
2. Создайте новый проект
3. Скопируйте URL проекта и anon key
4. Добавьте их в файл `.env`
